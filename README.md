Цель проекта: предварительная подготовка и сборка базовых компонентов для будущей разработки роботизированного манипулятора.

1.1 Проблематика существующих решений 

 • Ограниченная мобильность стационарных манипуляторов: Классические промышленные руки закреплены на одном месте, что сужает их рабочую зону до длины своего захвата.<br>
 •	Низкая доступность и высокая сложность: Сложные программируемые роботы имеют высокую стоимость и требуют специальных знаний для настройки и эксплуатации.<br>
 •	Негибкость управления: Многие системы имеют жёстко заданные алгоритмы и не позволяют оперативно адаптироваться к нестандартным задачам или удалённому управлению.<br>
 
 1.2 Преимущества проекта 6-осевой сервоприводной руки с дистанционным управлением
 
 •	Универсальность и доступность: Использование стандартных компонентов (Arduino, сервоприводы) делает сборку и модификацию доступной для образовательных целей, хобби и прототипирования.<br>
 •	Гибкость управления: Возможность прямого пошагового управления каждым суставом (через Bluetooth) позволяет выполнять как простые, так и сложные манипуляции, адаптируясь к задаче в реальном времени.<br>
 •	Потенциал для автономности: Базовая платформа (контроллер Arduino, датчики) позволяет развивать проект в сторону программирования и записи последовательностей движений.<br>
 •	Образовательная ценность: Проект наглядно демонстрирует принципы робототехники, кинематики, микроконтроллерного программирования и беспроводной связи.<br>
 
2. Сценарии применения

2.1	Образовательный демонстратор и тренажёр:<br>
•	Изучение основ робототехники, механики и программирования.<br>
•	Визуализация принципов работы промышленных манипуляторов.<br>
•	Платформа для экспериментов с алгоритмами управления и компьютерным зрением (при доработке).<br>
2.2	Прототип помощника для манипуляций в недоступной среде:<br>
•	Моделирование операций по переносу небольших объектов в условиях, где присутствие человека нежелательно или невозможно <br>
•	Дистанционный сбор предметов.<br>
2.3	Базовый модуль для более сложных роботизированных комплексов:<br>
•	Рука может быть установлена на мобильную платформу (робот-тележка), создавая аналог гибридной системы доставки, но для операций внутри помещения.<br>


3. Технические требования

Степени свободы:	6 осей: основание, плечо, локоть, запястье (2 оси), захват.<br>
Приводы:	6 сервоприводов + шаговый двигатель,	2 сервопривода синхронно работают на локтевом суставе для увеличения момента. Шаговый двигатель вращает основание.<br>
Система управления:	Микроконтроллер Arduino (Uno/Nano) + драйвер PCA9685,	PCA9685 обеспечивает точное управление несколькими сервоприводами по шине I2C.<br>
Связь:	Bluetooth (HC-05/HC-06),	дистанционное управление со смартфона или ПК.<br>
Программная логика:	Пошаговое управление суставами,	каждой команде (букве) соответствует движение конкретного сустава на фиксированный угол (increment).<br>
Питание:	Внешний источник 5-6В, 	необходимо для стабильной работы всех сервоприводов одновременно.<br>


5. Алгоритм выполнения задачи (на примере дистанционного управления)<br>
	 1 Инициализация системы: Подача питания, установка сервоприводов в "парковочную" позицию.<br>
	 2  Установка соединения: Подключение пульта управления (смартфона с терминалом Bluetooth) к модулю HC-05.<br>
	 3 Цикл управления:<br>
•	Оператор отправляет буквенную команду (например, 'C' для поднятия плеча).<br>
•	Контроллер Arduino принимает команду через последовательный порт.<br>
•	Вызывается соответствующая функция движения (shoulderServoBackward()).<br>
•	Функция проверяет, не достигнут ли механический предел, и отправляет новый сигнал на драйвер сервопривода.<br>
•	Сервопривод поворачивается на заданный угол, текущая позиция обновляется в переменной.<br>
•	Рука готова к приёму следующей команды.<br>
    4 Завершение работы: Возврат в "парковочную" позицию (при необходимости) и отключение.<br>
    
6. Пульт управления (Интерфейс оператора)<br>
Управление реализовано через терминал Bluetooth, что является гибким и минималистичным решением.<br>
•	Аппаратная часть: Стандартный модуль Bluetooth HC-05/06, подключённый к UART пинам Arduino.<br>
•	Программный интерфейс: Любое приложение-терминал на смартфоне или ПК (например, Arduino Bluetooth Controller, Serial Bluetooth Terminal).<br>
•	Схема управления (сопоставление команда-действие):<br>
•	S / O → Вращение основания влево / вправо (шаговый двигатель).<br>
•	c / C → Опускание / поднятие плеча (сервопривод 2).<br>
•	p / P → Опускание / поднятие локтя (синхронная работа сервоприводов 0 и 1).<br>
•	G / U → Наклон запястья 1 вниз / вверх (сервопривод 3).<br>
•	L / R → Вращение запястья 2 против / по часовой стрелке (сервопривод 4).<br>
•	f / F → Закрытие / открытие захвата (сервопривод 5).<br>
Для удобства в терминальном приложении можно создать кнопки-макросы, каждая из которых отправляет свою букву.<br>
